# 📘 EduCalc – GPA & CGPA Calculator

EduCalc is a simple and user-friendly web application designed to help students calculate their GPA and CGPA easily and accurately. It allows users to store academic records securely and track their performance semester-wise.

---

## ✨ Features

- Calculate GPA and CGPA instantly  
- Secure login using register number and password  
- Save and manage semester-wise records  
- Clean and responsive user interface  
- Reduces manual calculation errors  

---

## 🛠️ Technologies Used

- HTML  
- CSS  
- JavaScript  
- Firebase (for authentication and data storage)

---

## 🚀 How to Use

1. Open the EduCalc website.
2. Register using your register number and password.
3. Log in to your account.
4. Enter subject grades and credits.
5. Click calculate to view GPA/CGPA.
6. Save your results for future access.

---

## 🎯 Purpose

EduCalc helps students monitor their academic performance, set goals, and improve results by providing accurate and secure calculations.
